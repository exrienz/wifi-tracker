CSV File Requirements:

Your CSV file must contain the following columns (case-sensitive):

  • bssid - MAC address of the access point (e.g., AA:BB:CC:DD:EE:FF)
  • ssid - Network name
  • quality - Signal quality percentage (0-100)
  • signal - Signal strength in dBm (e.g., -45)
  • channel - WiFi channel number
  • encryption - Security type (e.g., WPA2, WEP, Open)
  • timestamp - Scan time (YYYY-MM-DD HH:MM:SS format)
Duplicate entries (same BSSID+SSID combination) will be automatically skipped.
Maximum file size: 1MB
